The Lantern Index is an invented project for cataloguing descriptions that use
different names for similar objects. Its demonstration record keeps the
original wording beside each normalised term.

## Intended outcome

The fictional project asks readers to compare classifications without hiding
their sources. It treats disagreement as useful information.

## Status

Demonstration only. No software, organisation, collection, or participant
described here exists.
