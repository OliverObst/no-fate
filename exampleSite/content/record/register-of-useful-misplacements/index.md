The register preserves the useful relationship between an object's expected
location and the place where it was eventually found.

| Ref. | Object | Expected | Found | Disposition |
| --- | --- | --- | --- | --- |
| R-04.01 | Brass key | Key board | Tea tin | Label retained |
| R-04.02 | Tide gauge pencil | Instrument desk | Left boot | Sharpened |
| R-04.03 | Red thread | Map drawer | Barometer case | Returned |
| R-04.04 | Small bell | Front counter | Archive box 7 | Left ringing |
| R-04.05 | Cupboard plan | Plan cupboard | Noticeboard | Annotated |
| R-04.06 | Noticeboard pin | Noticeboard | Cupboard plan | Reunited |

## Method

Entries are added only after two people have looked in the obvious place. The
original description is retained even when the object later acquires a better
name.

## Filing note

This register is complete enough to be useful and incomplete enough to remain
honest. Amendments should name the object, the mistaken assumption, and the
cupboard that benefited.
